# Node API - Primeiros passos no Backend

Este projeto faz parte dos meus primeiros estudos em desenvolvimento backend com Node.js. A ideia foi entender na prática como criar uma API simples, lidar com requisições HTTP e conectar uma aplicação a um banco de dados.

## Tecnologias utilizadas

* Node.js
* Express.js
* MongoDB Atlas
* Mongoose

## O que aprendi

Neste projeto, consegui:

* Criar um servidor básico com Express
* Trabalhar com rotas GET e POST
* Conectar a aplicação com um banco de dados MongoDB
* Criar um model com Mongoose
* Salvar dados enviados pelo cliente no banco
* Testar requisições usando Insomnia

## Como funciona

A API permite enviar dados através de uma requisição POST, que são armazenados no banco de dados. A estrutura inclui:

* title
* description
* image_url
* trailer_url

## Objetivo

Esse projeto representa meus primeiros passos no backend, focando em entender os conceitos fundamentais antes de evoluir para estruturas mais completas e boas práticas de desenvolvimento.
